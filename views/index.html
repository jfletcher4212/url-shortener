URL Shortener Microservice

User Stories:

  • The user can pass a URL as a parameter and receive a shortened URL in the JSON response.
  • Example Input: https://jfletcher-urlshortener.glitch.me/new/https://glitch.com/
  • When the user visits said shortened URL, it will redirect to the original link.
  • Example Input: https://jfletcher-urlshortener.glitch.me/goto/920f69f99
  • If an invalid URL or nonexistant shortened URL is given, an error will be returned.
